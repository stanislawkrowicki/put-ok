\doxysubsection{Detailed Description}
Implements \mbox{\hyperlink{classDijkstra}{Dijkstra}}\textquotesingle{}s algorithm for finding the shortest paths from a source vertex to all other vertices in a given graph. 

This class provides methods to run \mbox{\hyperlink{classDijkstra}{Dijkstra}}\textquotesingle{}s algorithm, retrieve the shortest distance to a specific vertex, and print the solution.

\begin{DoxyNote}{Note}
The graph is represented using an adjacency list. 
\end{DoxyNote}
